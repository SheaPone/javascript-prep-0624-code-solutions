## Quiz Questions

Answer the following questions in the provided markdown file before turning in this exercise:

- What are the five parts of a function definition?

1. function keyword
2. optional name
3. comma separated list of 0 or more parameters surrounded by ()
4. start of function code block {
5. an optional return statement
6. end of function code block }

- How do you call a function?

function Name();

- What is the difference between a parameter and an argument?

Parameter is a local variable to the function
An argument is a parameter when a function is called
